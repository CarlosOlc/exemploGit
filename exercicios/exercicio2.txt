Interface para Formas Geométricas

- Implemente uma interface chamada FormaGeometrica com os métodos abstratos:
    - calcular_area(): Método para calcular a área da forma.
    - calcular_perimetro(): Método para calcular o perímetro da forma.

- Crie classes concretas Circulo e Quadrado que implementem essa interface, com os métodos específicos para cada forma.
